# Privacy Policy - Sleepless Radio Discord Bot

**Effective Date:** October 7, 2025  
**Last Updated:** October 7, 2025

## 1. Introduction

This Privacy Policy describes how Ascend ("the Bot", "we", "us") collects, uses, and protects your information when you use our Discord bot services.

## 2. Information We Collect

### 2.1 Discord Information
- **User ID:** Unique Discord identifier (numerical)
- **Username:** Discord username for identification
- **Display Name:** Server-specific display name
- **Server ID:** Guild identifier where Bot is used

### 2.2 Account Information
When you create an Ascend account:
- **Account Creation Date:** Timestamp of account creation
- **Last Activity:** Last time you used bot commands
- **Command Usage:** Count of total commands used
- **Premium Status:** If applicable, premium subscription details

### 2.3 Spotify Integration Data
When you link your Spotify account:
- **Spotify User ID:** Your Spotify account identifier
- **Display Name:** Your Spotify display name
- **Email:** Your Spotify account email (if provided)
- **Country:** Your Spotify account country
- **Follower Count:** Number of Spotify followers
- **Access Tokens:** Encrypted tokens for Spotify API access
- **Refresh Tokens:** Encrypted tokens for maintaining access

### 2.4 Usage Data
- **Command Logs:** Commands executed and timestamps
- **Music Requests:** Songs requested and playback history
- **Queue Data:** Current music queue contents
- **Settings:** User preferences and bot configuration

### 2.5 Technical Data
- **Error Logs:** Technical issues for debugging purposes
- **Performance Metrics:** Bot usage statistics (anonymized)

## 3. How We Use Your Information

### 3.1 Core Functionality
- **Bot Operation:** Provide music streaming and bot services
- **Account Management:** Maintain user accounts and preferences
- **Music Integration:** Connect with Spotify and other music services
- **Command Processing:** Execute user requests and commands

### 3.2 Service Improvement
- **Bug Fixes:** Identify and resolve technical issues
- **Feature Development:** Improve existing features and add new ones
- **Performance Optimization:** Enhance bot speed and reliability

### 3.3 User Support
- **Technical Support:** Help resolve user issues
- **Account Recovery:** Assist with account problems
- **Feature Guidance:** Provide help and documentation

## 4. Data Storage and Security

### 4.1 Storage Location
- **Database:** SQLite database stored securely on our servers
- **Encryption:** Sensitive data (tokens, passwords) is encrypted
- **Access Control:** Limited access to authorized personnel only

### 4.2 Security Measures
- **Encryption in Transit:** All data transmission is encrypted
- **Secure Storage:** Database files are stored with restricted access
- **Regular Backups:** Data is backed up regularly for recovery
- **Access Logging:** All database access is logged and monitored

### 4.3 Data Retention
- **Active Accounts:** Data retained while account is active
- **Inactive Accounts:** Data may be deleted after extended inactivity
- **Deleted Accounts:** Data permanently removed within 30 days
- **Legal Requirements:** Some data may be retained as required by law

## 5. Data Sharing and Disclosure

### 5.1 Third-Party Services
- **Spotify:** Minimal data shared for music integration functionality
- **Discord:** Standard bot interaction data as required by Discord API
- **No Other Sharing:** We do not share data with other third parties

### 5.2 Legal Compliance
We may disclose information if required to:
- **Comply with Laws:** Meet legal obligations and court orders
- **Protect Rights:** Defend our rights and property
- **Prevent Harm:** Protect user safety and prevent illegal activities
- **Business Transfers:** In case of merger, acquisition, or asset sale

### 5.3 No Sale of Data
- **We never sell user data to third parties**
- **We never use data for advertising purposes**
- **We never share data for marketing purposes**

## 6. Your Rights and Choices

### 6.1 Data Access
- **View Account Data:** Use bot commands to view your stored information
- **Export Data:** Request data export in machine-readable format
- **Data Inquiry:** Contact us for detailed information about your data

### 6.2 Data Control
- **Update Information:** Modify account details using bot commands
- **Delete Account:** Permanently delete your account and all associated data
- **Spotify Unlinking:** Remove Spotify integration at any time
- **Settings Control:** Modify privacy and bot settings

### 6.3 Communication Preferences
- **Bot Notifications:** Control bot announcement and update messages
- **Direct Messages:** Opt out of bot direct messages
- **Error Reports:** Choose whether to include data in error reports

## 7. Cookies and Tracking

### 7.1 No Web Cookies
- **The Bot does not use web cookies**
- **OAuth callbacks use temporary session data only**
- **No persistent web tracking occurs**

### 7.2 Discord Tracking
- **Standard Discord API:** Uses Discord's built-in user identification
- **Command Logging:** Tracks commands for functionality only
- **No Cross-Platform Tracking:** Limited to Discord interactions
